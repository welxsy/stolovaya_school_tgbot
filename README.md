# stolovaya_school_tgbot
 **TELEGRAM BOT FOR THE SCHOOL CAFETERIA**
 
  Анализируя процесс организации питания в школьной столовой,  можно выявить несколько проблем. И  главная из них – сбор информации. Сбор списков людей, для которых накрывают еду в столовой, но проблема в том, что списки составляют сами ученики и они всегда не точны и составляются не всегда вовремя. Поэтому этот процесс следует автоматизировать. Достаточно доступный и простой вариант – создание чат-бота в Telegram.
Процесс разработки будет состоять из нескольких этапов:
 1.	Регистрация и создание самого бота.
 2.	Програмирование его базового функционала.
 3.	Создание базы данных для хранения списков учеников.
 4.	Разработка конечного функционала – создание Excel списков и автоматизация их отправки напрямую персоналу столовой.

- Выбор класса. На данном этапе выполняется SQL-запрос и полученные данные передаются пользователю в виде inline-кнопок, которые в свою очередь появляются под сообщением

- Каждый ученик отображается в виде кнопки с фамилией и именем.  Так же бот добавляет следующие ReplyKeyboardMarkup кнопки на главный экран пользователя: “ПРОСМОТРЕТЬ ТЕКУЩИЙ СПИСОК”, “РЕДАКТИРОВАТЬ СПИСОК”, “СОХРАНИТЬ И ОТПРАВИТЬ СПИСОК”, “Назад”.

<img width="666" height="423" alt="image" src="https://github.com/user-attachments/assets/46b16fae-6ed3-4dc7-a3ca-ffc1653121b6" />

- Далее бот обрабатывает кнопку выбора ученика и добавляет выбранных пользователем во временный список. 
- При нажатии “ПРОСМОТРЕТЬ ТЕКУЩИЙ СПИСОК” бот отправляет пользователю временный текущий список.

<img width="536" height="526" alt="image" src="https://github.com/user-attachments/assets/edbd9aae-a139-48ea-8c88-5ec5928a95b3" />
